# Event Management System

## Description
A role-based web application for managing memberships in an Event Management System.

## Features
- Role-based login (Admin / User)
- Session handling using localStorage
- Admin access to Maintenance module
- Add Membership
- Update / Extend Membership
- Cancel Membership (Checkbox logic)
- Reports module
- Transactions module
- Duplicate membership prevention
- Form validation

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)
- localStorage for data persistence

## How to Run
1. Download the repository.
2. Open the project folder.
3. Open `index.html` using Live Server (VS Code recommended).
